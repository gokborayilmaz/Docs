Upsonic Client is based on the requests and some other libraries.It provides a stable and compatible system for communication and utilization within the Upsonic system. In this document, we will guide you through the installation process of the Upsonic Client.

To get started, ensure you have:

- The **Python => 3** installed and operational
- A functional pip3 or pip command linked to your **Python => 3** installation

# Install
You can install the Upsonic Client with a single **pip** command:
```console
pip install upsonic
```
