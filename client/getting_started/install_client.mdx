Upsonic Client is based on the requests and some of other libraries. Its give an stable and compatiple system to communicating and using Upsonic systems. In this document we will install the Upsonic Client.

To get started, you will need:

- The **Python => 3** installad and working
- Ready to use pip3 or pip command that connects to your **Python => 3** installation

# Install
We will install in one line command with **pip**:
```console
pip install upsonic
```
