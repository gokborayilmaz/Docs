## Introduction
Upsonic Client is communicate with Upsonic On-Prem API and this communication is needs to a access key for authorization. For this we need to get `address` and `access_key` to connecting to API. The dashboard is giving it as well so we just need to copy and pastle.


## Connection
Please copy the connection code from **Dashboard**. Your connection code should like that:
```python

from upsonic import Upsonic_On_Prem
upsonic = Upsonic_On_Prem('https://your-server-address:5000', 'ACK_****************')
                
```
