## What is Upsonic ?
Upsonic gives teams instant access to shared code and frequently used snippets, preventing duplication of similar code and reducing spaghetti code creation.

<CardGroup cols={2}>
```python Sender
def sum(a, b):
    return a + b

upsonic.dump("math.sum", sum)

```
```python Receiver
math = upsonic.load_module("math")


math.sum(1, 2)
```
</CardGroup>


- Instant acccess to shared code
- Reduce spaghetti code creation
- Automatic documentation with ai
- Elastic dependency, you can use different versions of an library in same python envinronment
